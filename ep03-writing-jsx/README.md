# Episode 03 — Writing Markup with JSX

From [Run the Docs: React](https://run-the-docs.pages.dev/) video series.

## What you'll learn

- Why React mixes markup with rendering logic
- The three rules of JSX:
  1. Return a single root element (use `<>` Fragments)
  2. Close all tags (`<img />`, `<br />`, `<li>...</li>`)
  3. camelCase attributes (`className`, `onClick`, `strokeWidth`)
- Exceptions: `aria-*` and `data-*` keep their dashes

## Files

| File | Description |
|------|-------------|
| `TodoList.jsx` | Valid JSX with all three rules applied |
