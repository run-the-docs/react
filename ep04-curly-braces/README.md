# Episode 04 — JavaScript in JSX with Curly Braces

From [Run the Docs: React](https://run-the-docs.pages.dev/) video series.

## What you'll learn

- `"quotes"` for fixed strings, `{braces}` for JavaScript values
- Any expression works: variables, function calls, math, ternaries
- Use curly braces in text content and after `=` in attributes
- `{{ }}` = an object inside curly braces (not special syntax)
- Inline styles use camelCase: `backgroundColor`, `fontSize`
